WINDOWS LIVE CD FR - Un Windows sur un fichier ou votre clé USB !


Beaucoup de Lives CD de Windows existent, Hiren Boot CD, AdhvBoot, Windows 10 PE Bobomb, et bien plus encore.
Cependant je n'ai jamais vu de Live CD Français, je vais essayer d'y remédier.

Le but d'un Live CD est un système d'exploitation en un fichier bootable/démarable contenant des logiciels de dépanage avec des permissions administrateurs les plus ouvertes possibles.

Je ne cautionne en AUCUN CAS : Ceux qui utilisent cet ISO à des fins criminelles (Cambriolages / Remote Acces Trojan R.A.T), je renie ces personnes !
